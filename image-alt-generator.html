Alt Text Best Practices

  • Keep alt text under 125 characters for optimal screen reader compatibility
  • Be descriptive but concise - describe what's in the image
  • Include relevant keywords naturally, don't keyword stuff
  • Don't start with "Image of..." - screen readers already announce it's an image
  • Use empty alt="" for purely decorative images
  • Include context about what the image shows in relation to the page content
