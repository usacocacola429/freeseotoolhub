Optimization Guide 2026

How to Improve INP Score

  • Minimize Main Thread Work: Break up long tasks (>50ms).
  • Optimize Event Callbacks: Debounce scroll and resize handlers.
  • Use Web Workers: Offload heavy JS to background threads.
  • Reduce DOM Size: Keep it under 1500 nodes if possible.
  • Yield to Browser: Use `setTimeout` or `scheduler.yield()`.
