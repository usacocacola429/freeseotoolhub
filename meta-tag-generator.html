Meta Tags Best Practices
Title Tag
  • Keep between 50-60 characters
  • Include primary keyword near the beginning
  • Make it compelling and click-worthy
  • Use pipe (|) or dash (-) as separators
Meta Description
  • Optimal length: 150-160 characters
  • Include primary and secondary keywords
  • Write a compelling call-to-action
  • Make it unique for each page
Open Graph Tags
  • og:title: 60-90 characters
  • og:description: 150-200 characters
  • og:image: 1200×630px (1.91:1 ratio)
  • Always include og:url and og:type
Twitter Cards
  • Choose between summary and summary_large_image
  • Image size: 1200×600px (2:1 ratio)
  • Include twitter:site for branding
  • Test with Twitter Card Validator
